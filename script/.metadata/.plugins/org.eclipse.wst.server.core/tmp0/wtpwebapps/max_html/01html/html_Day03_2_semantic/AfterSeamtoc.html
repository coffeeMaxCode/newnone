시맨틱 웹

			시맨틱 태그는 모두 div태그와 같은 기능 수행함
			검색엔진 or 기게적 동작들이 웹페이지에 쉽게 이해할 수 있도록 제공
			html5이전 : 화면레이아웃에 table 태그 주로 사용
			html5이후 : div태그 주로 사용 > ajax기술지원

HTML5 시맨틱 구조 태그

			header	: 헤더 의미
			nav		: 내비게이션 의미
			section	: 여러 중심 내용 감싸는 공간
			article : 글자가 많이 들어가는 내용
			footer	: 페이지 하단 부분
